# Alpha 网站展示规则

## 默认隐藏

- 名人、网红、普通个人账号、项目创始人个人号。
- 成熟知名项目、大公司、交易所、媒体、基金、做市商、招聘号。
- 纯营销号、抽奖号、互关号。
- 信息不足的 NFT / meme / 纯 CA 项目，除非出现强交叉信号。
- TG 与 X 中重复出现的同一项目只展示一条，不重复刷屏。

## 优先展示

- TG 和 X 双来源都出现的项目。
- 至少 2 个 Alpha 共同关注的项目。
- TG 当前窗口或历史累计出现至少 2 次的项目。
- 有明确产品方向的早期项目，例如：
  - AI Agent
  - agent payment
  - Uniswap v4 hooks
  - launchpad
  - onchain game
  - crypto infra
  - Bitcoin L1
  - Solana app
  - Base app

## 展示格式

- 同一项目按 X handle 合并。
- 表格优先显示：项目、X 账号、来源、Alpha 数、TG 次数、用途、早期信号。
- 成熟但被多次关注的项目只放备注，不进入重点列表。
